# 4. 管理画面とテスト URL 確認

## 概要

Milestone 4 では、会員向けの予約機能だけでなく、運営管理に必要な管理画面も整備いたしました。

管理者さまは、イベントの作成、定員設定、公開・非公開管理、予約一覧確認、代理予約、予約変更、キャンセル、履歴確認までを管理画面から行えます。これにより、見学会・個別見学の受付業務をシステム上で一元管理できる状態になりました。

| イベント一覧 | 完了 | 見学会・個別見学を一覧で確認 |

## イベント管理

`/admin/events` では、登録済みのイベントを一覧で確認できます。

管理できる主な項目は以下です。

- イベント種別
- イベント名
- 開催日時
- 終了日時
- 定員
- 開催場所
- 支援者限定
- 公開状態
- 説明文

イベント詳細画面 `/admin/events/[id]` では、定員、予約席数、残席を確認しながら、イベント情報を編集できます。参加者一覧と履歴も同じ画面で確認できるため、イベント単位で運用状況を把握しやすい構成にしています。

## 予約管理

`/admin/bookings` では、イベントを選択して予約一覧を確認できます。

管理者さまは、以下の操作を行えます。

- 会員検索による代理予約
- 予約人数の変更
- 予約メモの変更
- 予約状態の変更
- 予約キャンセル
- 必要時の予約削除

代理予約では、会員名・メールアドレスによる検索を利用できるため、UUID のような内部 ID を直接入力する必要はありません。現場運用での入力ミスを減らすため、顧客検索 API と連動したフォームを実装しています。

## 監査ログ

`/admin/audit-logs` では、予約・イベントを含む操作履歴を一覧で確認できます。

主な確認項目は以下です。

- 操作日時
- 操作者
- アクション
- 対象テーブル
- 対象 ID
- 詳細メタ情報

アクションや対象テーブルで絞り込めるため、予約関連の問い合わせやイベント設定変更の確認にも利用できます。

## テスト URL

貴社にご確認いただくためのテスト URL は以下です。

- https://horse-system-mu.vercel.app/

この URL から、会員マイページ、管理画面、予約機能、支援・決済関連画面をご確認いただけます。

主な確認ポイントは以下です。

- 会員としてログインし、見学会・個別見学の一覧を確認できること
- 予約可能なイベントで予約できること
- 自分の予約を変更・キャンセルできること
- 管理者としてイベントを作成・編集できること
- 管理者として予約一覧を確認できること
- 管理者として代理予約・変更・取消ができること
- 監査ログに操作履歴が残ること
